# WoopraTrack

Tracking library for woopra.com. Woopra analytics client-side and server-side tracking helper.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'woopra_track'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install woopra_track
